# Gas Station Inventory Management

This project is a backend application for managing gas station inventory using OCR to scan and process invoices.

## Features

- Upload and process invoice images.
- Extract text from invoices using OCR.
- Manage inventory data based on extracted information.

## Technologies Used

- Node.js
- Express.js
- Together AI API
- Multer for file uploads
- Axios for HTTP requests
